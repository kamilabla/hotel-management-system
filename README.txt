Program obsługujący zarządzanie hotelu.
Składa się z następujących plików:
    * Main.java:
        Główna klasa, która po uruchomieniu programu pobiera następujące komendy;
        checkin - zameldowanie gościa
        checkout - wymeldowanie gościa
        list - wylistowanie wszystkich dostępnych pokoi wraz z informacjami o numerze pokoju,
        imieniu i nazwisku gościa oraz piętrze, na którym znajduje się dany pokój
        view - pokazanie informacji o danym pokoju
        save - zapis informacji o wszystkich pokojach do pliku CSV
        exit - wyjście z programu

    * MyMap.java
    * Room.java
